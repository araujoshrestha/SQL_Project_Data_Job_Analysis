**📊 SQL Job Market Analysis (PostgreSQL Project)**
Overview: 
This project explores real-world job market data using SQL (PostgreSQL) to uncover insights about:
- High-paying data roles
- Most in-demand skills
- Skills that lead to higher salaries
- The intersection between demand and compensation
The goal is to understand what skills are most valuable in today’s data job market, and build a strong portfolio project for data analyst roles.

1) Which data-related jobs pay the most?
